HTML (HyperText Markup Language) es un lenguaje de marcado utilizado para la creación y estructuración de páginas web. No es un lenguaje de programación, sino un conjunto de etiquetas (tags) que permiten organizar el contenido en distintos elementos como texto, imágenes, enlaces, tablas y formularios.

Etiqueta de estrucura: <html>,<headl>,<bodyl>

Etiqueta de contenido: <p>,<h1>,<img>
